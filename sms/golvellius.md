# Golvellius: Valley of Doom 

## RAM Locations

| Address | Size | Description                                              |
| ------- | ---- | -------------------------------------------------------- |
| $081A   |    1 | Current player health. Range 0-6                         |
| $0836   |    1 | Potion count, unused                                     |
| $0839   |    1 | Potion count, maximum                                    |
